## Getting Started
To get a local copy of the app, run the following commands in your terminal:
- `git clone https://github.com/yusufnviiri/costs`
- `cd /costs`
- `bundle install`

Run the following commands to run the app

- `rails db:create`
- `rails db:migrate`
- `rails s`

In your browser, go to

- `http://localhost:3000`
